# Markdown



## 1.标题

## 语法

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

```markdown
***粗体并斜体***
___粗体并斜体___
~~加删除线的文字~~
```

## 7.列表

### 语法

~~~ markdown
- 内容
+ 内容
* 内容
- 1.列表内容
2.列表内容
3.列表内容

上一级和下一级之间三个空格
~~~

- 列表一
- 列表二
  - 1.列表内容
    - 2.列表内容
      - 3.列表内容
